# All Links from given Webpage

This script extracts all links from a given Webpage

### Prerequisites
Required Modules
- BeautifulSoup4
- requests

to install: 
```
$ pip install -r requirements.txt
```

### How to run the script
``` bash
$ python Web-Link Extractor.py
```
